Knowledge Base

The durable memory of this project. Read this before picking up work. Chat history is not the record; this is.

Mission Control is a local-first command center to watch (and act on) the activity across the dev stack and AI agents (Claude Code + Codex). The plan of record is PLAN.md; the repo is umarali/mission-control.

How to use this

  • New here (human or agent)? Read the entries below in order, then PLAN.md. Ask for /teach if you want a guided walkthrough.
  • Each entry is structured Problem → Solution → Context: what the problem was and why, what was built and why that way (decisions + edge cases), and why it matters / what it impacts.
  • Quality bar: comprehensive but brief. The why and the decisions, not a transcript.

When to add or update an entry

Before handing off to a new agent, before starting a new task, or when an agent reaches ~60–65% of its context window. Run /handoff to update the base and produce a paste-ready handoff prompt. Start a new entry from _template.html.
